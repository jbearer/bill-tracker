Module model::graphql::types

source ·
Expand description

Types and traits describing the GraphQL protocol.

These types constitute the Rust bindings to the generic GraphQL protocol. They form the foundation upon which our specific GraphQL API is built.

Many of the types in this module are reexports from the [async_graphql] crate.
