Expand description

The base type of the whole GraphQL type system.

All objects in a relational GraphQL API, from scalars to copmlext object types, have a type which is a subtype of Type, and this trait describes the functionality which is common to every entity in an API.

Required Associated Types§

source

type Predicate: Predicate<Self>

A boolean predciate on objects of this type.

source

type PluralPredicate: PluralPredicate<Self>

A predicate on collections of objects of this type.

source

type Input: Scalar

The representation of values of this type when creating a new object.

For scalars, the input is just the value of the scalar itself. For resources, the input is the ID of the referenced resource.
