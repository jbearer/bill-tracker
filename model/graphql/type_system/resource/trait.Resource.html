fn build_resource<B: ResourceBuilder<Self>>(
    builder: B
) -> Result<Self, B::Error>

Build a resource using a builder supplied by the backend.

This performs the same operation as build, but it can be called directly with a ResourceBuilder, instead of the more generic Builder. This is useful when it is known that a Type is actually a Resource.

It is an invariant that for all T: Resource, T::build(builder) == T::builder_resource(builder.resource()).

Describe the structure and definition of this Resource.
