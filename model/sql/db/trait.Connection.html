fn select<'a>(
    &'a self,
    columns: &'a [SelectColumn<'a>],
    table: impl Into<Cow<'a, str>> + Send
) -> Self::Select<'a>

Start a SELECT query.

columns indicates the columns to include in the query results. The resulting Select represents a statement of the form SELECT columns FROM table. The query can be refined, for example by adding a WHERE clause, using the approriate methods on the Select object before running it.
